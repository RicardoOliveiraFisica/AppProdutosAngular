Sobre

Este aplicativo é o resultado final de um curso de treinamento de Angular/Java ministrados pelos professores Eduardo, Gleyser e Rafael da Minsait


O aplicativo foi construído com:

  • Angular 15.2.4
  • Bootstrap 5.3.2
  • Sweetalert2 11.8.0
  • Material 15.2.9

Este aplicativo consome dados do aplicativo de backend em Java que acessa um banco MySQL
